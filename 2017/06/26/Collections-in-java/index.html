Collections in java

Posted on 2017-06-26 | Edited on 2019-01-09 | In java |

Collections framework in java language is a very important tool. We can use it to arrange

many items together.

What is it

A collections framework is a unified architecture for representing and manipulating collections.

It includes three ingredients: interface, implementation and algorithm. Its interface defines the

operations, the implementation implements it, and the algorithm includes its core computation.

Interfaces and implementations

There are many interfaces in them, but there are main five interfaces. They are set, list, queue,

deque and map. All of them extend collection interface, so they can use the basic operations in

collection. But these specific interfaces have more powerful.

Set is a collection that does not contain same item. That is, each item of them must not be the

same. HashSet, TreeSet and LinkedHashSet are children of it.

List is a collection that is sequenced in a line, or called sequence. It arrange items in a row,

that is, each item in it can be access by the position. ArrayList and LinkedList are children of it.

Queue adds an item from an end, and delete an item from the other end. So, it behaves the FIFO. You

can find its children in java.util.concurrent package, such as BlockingQueue.

Deque can add and delete items from both ends which is different from queue. Its children are made

up of ArrayDeque and LinkedList.

Map is another collection that is not the child of the collection. It contains keys and values.

Each key is unique and the key collection is a set in essence. Just like the set, we find HashMap,

TreeMap and LinkedHashMap are direct children of it.

Algorithms

Though these interfaces or implementations are different in specific algorithms, they share some common

methods, such as sort, shuffle, swap, add, delete and the like.

Above all are the brief introducation about the collections framework in java. You can use these implementations

directly or implement your algorithms if they can not meet your demands.
